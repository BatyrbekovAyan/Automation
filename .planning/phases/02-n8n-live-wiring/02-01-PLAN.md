## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → `/webhook/SuggestReplies` | Untrusted, UNauthenticated HTTP request enters the workflow (no API key, by design). |
| customer message content → LLM prompt | `messages[].text` + `lastIncomingText` are attacker-controllable (a customer typed them) and flow into the model. |
| `botWaId` (client-supplied) → RAG filter | The client scopes retrieval; a wrong/wide value could cross tenants. |
| LLM output → response payload | Non-deterministic model text must never reach the client raw. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
